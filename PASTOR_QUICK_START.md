# Pastor Dashboard - Quick Start Guide

## Accessing the Dashboard

1. **Visit the app:** `https://your-app.up.railway.app`
2. **Open sidebar:** Click the ☰ menu icon (top-left corner)
3. **Select dashboard:** Click "📊 Pastor Dashboard"
4. **Login:** Enter the password provided by your administrator
5. **Review questions:** Browse, filter, and export submitted questions

## Dashboard Overview

### Main Sections

**📈 Overview**
- Total questions submitted
- Number of unique topics
- Average tone score
- Recent activity (last 7 days)

**🔍 Filters & Search**
- Filter by topic cluster
- Sort by date or tone score
- Search by keywords in questions/answers

**💬 Questions List**
- View all submitted questions
- See full question and answer text
- Review metadata (topic, timestamp, tone score)
- Take actions (approve, reject, tag)

**📥 Export Data**
- Download as CSV for Excel
- Download as JSON for analysis
- Export filtered results

**📊 Analytics**
- Topic distribution chart
- Tone score trends
- Questions over time

## Common Tasks

### Reviewing a Question

1. Scroll to question in list
2. Click to expand
3. Read question and answer
4. Check tone score (🟢 good, 🟡 neutral, 🔴 needs review)
5. Click action button:
   - **✅ Approve** - Mark for training data
   - **❌ Reject** - Exclude from training
   - **🏷️ Re-tag** - Change topic classification
   - **📝 Add Note** - Add pastor comments

### Filtering Questions

**By Topic:**
- Select topic from dropdown (e.g., "Salvation", "Baptism", "Prayer")
- Dashboard shows only questions in that category

**By Date:**
- Sort by "Newest First" or "Oldest First"
- Use analytics to see date ranges

**By Quality:**
- Sort by "Highest Tone Score" (best answers)
- Sort by "Lowest Tone Score" (needs review)

### Searching Questions

1. Enter keywords in search box (e.g., "baptism", "forgiveness")
2. Dashboard filters in real-time
3. Search looks in both questions and answers
4. Clear search to see all results

### Exporting Data

**For Excel Analysis:**
1. Apply desired filters
2. Click "Download as CSV"
3. Click "Download CSV File"
4. Open in Excel or Google Sheets

**For Backup:**
1. Click "Download as JSON"
2. Save file with date in filename
3. Store securely

## Understanding Metrics

### Tone Score

**Scale:** 0.0 to 1.0

- **🟢 0.7-1.0** - High quality, doctrinally sound, compassionate
- **🟡 0.4-0.6** - Neutral, adequate but may need refinement
- **🔴 0.0-0.3** - Low quality, may need pastoral review

**What it measures:**
- Theological accuracy
- Pastoral tone and compassion
- Clarity of explanation
- Alignment with WELS doctrine

### Topic Clusters

Questions are automatically categorized into topics:

- **Salvation** - Grace, faith, works, justification
- **Sacraments** - Baptism, Lord's Supper
- **Scripture** - Bible interpretation, authority
- **Prayer** - Communication with God, petitions
- **Worship** - Church practices, liturgy
- **Ethics** - Moral issues, Christian living
- **General** - Uncategorized or multi-topic questions

### Response ID

Unique identifier for each question (format: `YYYYMMDDHHMMSSMS`)

- Useful for tracking specific questions
- Reference when discussing with other pastors
- Helps locate questions in logs

## Best Practices

### Weekly Review

Set aside time each week to:

1. Review new questions (sort by "Newest First")
2. Approve high-quality answers
3. Flag problematic answers for discussion
4. Note emerging topics or trends

### Quality Checks

Look for:

- **Doctrinal accuracy** - Does answer align with WELS teaching?
- **Pastoral tone** - Is response compassionate and helpful?
- **Scripture references** - Are biblical citations accurate?
- **Completeness** - Does answer fully address question?
- **Clarity** - Is explanation understandable to layperson?

### Data Privacy

Remember:

- All questions are **anonymized** (PII removed)
- User IDs show "anonymous" (no personal tracking)
- Email addresses and phone numbers are redacted
- Safe to export for internal review
- Do not share exports publicly

### Security

- **Never share password** with unauthorized persons
- **Log out** when finished reviewing
- **Close browser** on shared computers
- **Report suspicious activity** to administrator

## Troubleshooting

### Can't access dashboard

**Problem:** Page not loading
**Solution:**
- Ensure you're logged into the app
- Try refreshing the page (F5 or Cmd+R)
- Clear browser cache

**Problem:** Password not working
**Solution:**
- Verify password with administrator
- Check for extra spaces when pasting
- Try typing password manually

### No questions appearing

**Problem:** Dashboard shows "No questions yet"
**Solution:**
- This means no users have asked questions
- Test by asking question in main chat
- Check if filters are too restrictive

### Charts not displaying

**Problem:** Analytics section blank
**Solution:**
- Requires at least 3 questions for charts
- Check browser JavaScript is enabled
- Try different browser (Chrome, Firefox)

### Export not working

**Problem:** Download button doesn't work
**Solution:**
- Check browser pop-up blocker
- Allow downloads from this site
- Try right-click → "Save Link As"
